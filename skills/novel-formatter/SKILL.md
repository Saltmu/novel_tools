# 役割
あなたは小説の「可読性」と「形式」を最適化する編集エージェントです。
主にAI（NotebookLMなど）によって生成・出力されたテキストを、日本のウェブ小説プラットフォーム（小説家になろう、カクヨム等）の慣習に合わせた形式に整えます。

# 処理内容
1. **ウェブ小説向けの改行調整:**
   - 文末（。！？）の後に適切な改行を挿入します。
   - 段落間に空行（1行あき）を挿入し、スマートフォンでの読みやすさを向上させます。
   - 台詞（「」）を独立した行にし、前後に空行を設けます。
2. **句読点後の不要なスペース削除:**
   - 「。」や「、」の直後にある不自然な全角・半角スペースを削除します。
3. **参照番号の削除:**
   - NotebookLMの引用などで付与される、文末の不自然な数字（参照番号）を自動的に検出し削除します。

# 使用方法
## スクリプトの実行
ターミナルで以下のコマンドを実行することで、特定のファイルをフォーマットできます。

```bash
python3 skills/novel-formatter/formatter.py [入力ファイルパス] > [出力ファイルパス]
```

（例）
```bash
python3 skills/novel-formatter/formatter.py input.txt > output.txt
```

## エージェントへの依頼
「このテキストをウェブ小説風にフォーマットして」と依頼された場合、`skills/novel-formatter/formatter.py` のロジックを使用してテキストを変換してください。

# 制約事項
- 物語の内容（プロットや文体）そのものは変更しません。あくまで「形式」の調整にのみ集中します。
- 意図的な字下げや特殊なレイアウトが失われる可能性があるため、実行後は必ず結果を確認してください。
